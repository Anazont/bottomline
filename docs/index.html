Arrays

compact(Array);

Creates an array with all falsey values removed.

The values false, null, 0, "", undefined and NaN are all falsey.

Arguments:
  • (Array): The array to compact.
Returns:
  • (Array): Returns a new array of filtered values.
Tag: @arrays @compact

Example:

$array = array(
	0, 3, 100, '', false, true
);

$result = __::compact($array);

var_dump($result);

difference(Array, Array);

Creates an array excluding all values of the provided arrays.

Using strict equality for comparisons, i.e. ===.

Arguments:
  • (Array): The array to process.
  • (Array): The arrays of values to exclude.
Returns:
  • (Array): Returns a new array of filtered values.
Tag: @arrays @difference

Example:

$first = array(
	10, 20, 30, 15, 5
);

$second = array(
	15, 65, 10
);

$result = __::difference($first, $second);

var_dump($result);

flatten(Array);

Flattens a nested array (the nesting can be to any depth).

Arguments:
  • (Array): The array to flatten.
Returns:
  • (Array): Returns a new flattened array.
Tag: @arrays @flatten

Example:

$array = array(
	array(3, 4, array(6, 3),
	array(6, array(2, array(2, 9))),
	11
);

$result = __::flatten($array);

var_dump($result);
